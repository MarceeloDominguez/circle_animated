## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/circle_animated.git
```

Go to the project directory

```bash
  cd circle_animated
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
